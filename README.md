## T-task

#### Download repository:
```sh
git clone https://github.com/Boris1979509/T-task.git
```
#### Go to the app:
```sh
$ cd T-task
```
#### Install dependencies:
```sh
$ npm install
```
#### Server with hot reload at http://localhost:8081/
```sh
$ npm run dev
```
#### Output will be at dist/ folder
```sh
$ npm run build
```
